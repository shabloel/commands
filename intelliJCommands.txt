ctr + shift + f		 	-> global search in Path
ctr + g					-> type lijnnummer in om code te vinden
ctr + r					-> rename
ctr + b					-> get to where a method or variable is declared

Code generation:
********************

ctr + Alt + O			-> organizes imports
ctr + Alt + v			-> extract variable
ctr + alt + m           -> extract a method
alt + enter				-> imports package or if your cursor is on class name you can create a test
ctr + alt + t			-> surrounds code with try and catch block

shift + ctr + enter		-> completes current statement
sout 					-> system.out.println
ctr + D					-> duplicate code fragment
ctr + y					-> delete line
